실행 설정:
컨트랙트들이 너무 복잡하게 얽혀있어서
필요한거(프론트에서 직접 접근하는 컨트랙트들) 외 정리했습니다
sol파일 관리는 openZepplin 설치해서 관리하고 있습니다
서버 올리기위해서 client 폴더로 이동 후 yarn install로 의존성 설치해주셔야 합니다.
 
내용 설명:
토큰을 판매/구매하는 페이지 입니다
/profile 에서 mint된 토큰을 해당 페이지에서 판매설정 할 수 있습니다
해당 페이지에서 모든사람이 판매가 개시된 토큰을 구매할 수 있습니다
구매시 수수료가 붙습니다 현재 초기값은 수수료율50% 입니다(조정가능)
수수료는 Broker.sol 파일의 uint256 commision 변수가 결정합니다
(ex. commision값이 2 = 수수료 50%, 값이 20이면 5%)

에러나는 경우
1. 판매자/ 구매자만이 접근가능한 함수에 올바른 계정으로 접근하고 있는지 확인해주세요
2. migrate --reset 해주세요
3. 가나시없이 메타마스크를 통해 계정 import 해서 돌릴때 RPC_ERROR 나는경우
메타마스크 설정 들어가서 계정 초기화 해주세요.

화면 설명:

tokenMint
URI 입력 후 버튼을 누르면 토큰 생성됨(test용)

tokenOwn
토큰 아이디 입력후 버튼 누르면 콘솔에 해당하는 토큰의 소유자가 나옴

tokenBlc
버튼 누르면 현재까지 발행된 토큰 수 보여줌

tokenURI
토큰id 입력후 누르면 해당하는 토큰의 URI 보여줌

toknSail
토큰id와 가격(반드시 정수값) 입력 후 누르면 해당 토큰을 입력가격에 판매 시작함
(해당 토큰 소유자만 접근가능)

toknClose
토큰 id 입력 후 누르면 해당 토큰의 판매 중지됨
(해당 토큰 소유자만 접근가능)

available
현재 판매중인 모든 토큰의 id를 콘솔에 표시함

tokenBuy
토큰 id입력 후 누르면 토큰 구입이 가능함
(판매중인 토큰만 구입 가능, 해당 토큰 판매자는 접근안됨)

CA_Balance
현재 컨트랙트가 보유하는 이더량을 표시함

withdraw
현재 컨트랙트가 보유한 이더를 모두 지정된 계좌로 옮김
(지정된 오너만 접근 가능함)
